You must have working OpenCV dlib and Numpy or install them from using pip

You also must have trained predictor which is provided in same directory


Note: Change the test.jpg to your image

How to run

1. Download all four file 
	face_landmarks.py                     
	readme.txt                            
	shape_predictor_68_face_landmarks.dat
	swami_vivekanand.jpg

2. Place them in a folder

3. Also place your image( default(swami_vivekanand.jpg) ) in same folder

4. run face_landmarks.py
